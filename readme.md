## Usage

```
$ 9z --help

	Usage
	  $ 9z [...]
	Options
	  --csgo        Show CS:GO
	  --fortnite    Show Fortnite
	Examples
	  $ 9z
	  # Prints everything
	  $ 9z --csgo
	  # Prints CS:GO only
	  $ 9z --csgo --fortnite
	  # Prints CS:GO and Fortnite only
```
